# Personal Site Goals

## High-value goals

この種の個人サイトが目指す価値は、単なる portfolio ではなく次の複合体だと考える。

1. owned identity
   - 自分の名前、文体、制作物、導線を自分の domain に集約する
2. durable memory
   - 流れて消える SNS より、後から見返せる記録と導線を育てる
3. living curation
   - 好きなもの、使うもの、考えたことを更新し続ける
4. utility for self
   - 自分が毎日使う dashboard や pointers を持つ
5. discoverable taste
   - 技術力だけでなく、何を良いと感じるかが伝わる
6. room for unfinished work
   - 完成品だけでなく、育成途中の考えも置ける

## Product implications

- blog だけでなく curated collection が重要
- pointers / links / recommendations のような「軽い更新」が価値になる
- admin friction は低くないと継続できない
- static speed は維持しつつ、更新導線だけを軽くするべき
- archive と garden の両面を持たせると長期価値が出る
